# Introduction to Git

## This repository

This repository is meant to be the git repo as used in a introductory training to git.

In order to use this;
1. Fork it
2. Allow people to clone _the fork_
3. People can then work on the assignments


## The assignments

1. Change the text that is presented `"Hello world"` -> `"Hello Git training"`
2. Add a route that yields a different message
3. Pull changes from the remote
4. Add a new router (multiple files)
5. Git reset: we've made a mistake
6. explore the .gitignore
    * We add a file to the project, but it is not uploaded, what!?
    * check the .gitignore

## Part two of the assignments

1. Checkout a new branch
2. Create a local merge (master into new branch)
3. Pull & merge conflict
4. Pull requests
    * Create a new branch
    * Make changes
    * Create a pull request
5. Git revert a commit
